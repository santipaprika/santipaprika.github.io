    Save the Zebra

    On-demand SOLO project

    Description

    Unity-based puzzle game prototype for a Fiverr customer in which the user has to open the given gates in the right order so that the zebra can arrive to the zebra’s food without having an open path to the panther, because in that case the panther will go and “eat” the zebra.
